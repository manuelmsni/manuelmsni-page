Para establecer el color de un elemento HTML, se le asigna una propiedad en CSS, en esta se especifica que el elemento es de dicho color.

CSS hace uso de la codificación RGB para establecer los colores. En la codificación RGB se emplea un byte para representar cada uno de los tres "colores primarios luz" (Red, Green y Blue). No obstante los navegadores también soportan el manejo de la transparencia de los elementos, para lo cual se reserva un byte adicional para el canal Alpha.

En notación RGB este canal se suele representar como un número de 0 a 1 (siendo 0 completamente transparente y 1 completamente opaco) o como un porcentaje de 0% a 100% (siendo 0% completamente transparente y 100% completamente opaco). En notación hexadecimal se añaden dos dígitos de 00 a ff (siendo 00 completamente transparente y ff completamente opaco).

Sin hacer uso del canal Alpha, podemos especificar el color blanco en RGB dentro de una propiedad CSS con diferentes notaciones:

  • rgb(255 255 255)
  • #ffffff

Si quisiéramos especificar el valor Alpha al 75% de opacidad del color blanco, podríamos hacerlo de diferentes formas:

  • rgb(255 255 255 / 75%)
  • rgba(255 255 255 / 0.75);
  • #ffffffbf

Podemos visualizar un ejemplo rápido en Chrome. Si en un elemento de la página web hacemos clic derecho y seleccionamos inspeccionar, a la derecha nos aparecen las "herramientas para desarrolladores", en estas podemos consultar el código CSS.

Si nos ubicamos en un fragmento de código en el cual se especifique un color en hexadecimal (símbolo "#" seguido de 6 dígitos o caracteres) y hacemos clic en el cuadrado que lo precede, se abre una pequeña interfaz que nos permite modificar el mismo sin necesidad de conocer la notación para la codificación RGB. Si en esta modificamos el canal alfa, podemos apreciar como al código en hexadecimal se le añaden dos caracteres extra (1 byte).
