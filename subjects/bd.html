¿Qué es "DDL"? ⬇️

El lenguaje de definición de datos (DDL) es un subconjunto de SQL. Es un lenguaje para describir los datos y sus relaciones en una base de datos.

Este lenguaje permite definir las tareas de las estructuras que almacenarán los datos.

Sentencias DDL son aquellas utilizadas para la creación de una base de datos y todos sus componentes: tablas, índices, relaciones, disparadores (triggers), procedimientos almacenados, etc.

  1. CREATE: Crear nuevas tablas, campos e índices.
  2. ALTER: Modifica las tablas agregando campos o cambiando la definición de los campos.
  3. DROP: Elimina tablas e índices.
  4. TRUNCATE: Elimina todos los registros de una tabla.
  5. COMMENT: Agrega comentarios al diccionario de datos.
  6. RENAME: Renombra objetos.
