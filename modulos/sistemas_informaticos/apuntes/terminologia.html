Pasar de binario a decimal

Por ejemplo, para pasar el número (1101001)2 a decimal:

1 1 0 1 0 0 1
26 25 24 23 22 21 20

1x26 + 1x25 + 0x24 + 1x23 + 0x22 + 0x21 + 1x20

64 + 32 + 8 + 1 = 105

(1101001)2 ≡ (105)10


Pasar de decimal a binario

Para pasar un número de decimal a binaro, se divide entre 2 sucesivamente. Encadenando el resto de cada operación se forma el número en binario.

Por ejemplo, para pasar el número (105)10 a binario:

(105)10 ≡ (1101001)2
