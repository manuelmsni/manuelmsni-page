Sistemas de información:

Conjunto de recursos físicos, humanos y de protocolo encaminados a gestionar y organizar información.

Componentes de un sistema de información:

  • Datos: Información relevante que se almacena y gestiona.
  • Hardware: Equipamiento físico que se emplea para gestionar los datos.
  • Software: Aplicaciones informáticas que se encargan de la gestión de la base de datos.
  • Recursos humanos: Personal que maneja el sistema de información.
Tipos de sistemas de información:
  1. Sistemas de información orientados al proceso
  2. Sistemas de información orientados a los datos

1 Sistemas de información orientados al proceso (sistemas de ficheros):

Emplean diversas aplicaciones. Cada aplicación almacena sus propios datos independientemente. La modificación de un dato en una aplicación no afecta al resto de aplicaciones.

Esto trae grandes inconvenientes:

  • Datos redundantes.
  • Datos inconsistentes: Si cambias el dato en una aplicación, el mismo dato podría quedar con distinto valor en otra aplicación.
  • Coste de almacenamiento elevado: Repetición de datos.
  • Difícil acceso: Si se requiere una consulta no prevista, hay que cambiar el código o crear una nueva aplicación.
  • Tiempos de prosesamiento elevado: Almacenamiento no optimizado.
  • Dificultad de acceso simultaneo: Dos usuarios no pueden acceder a los datos de forma concurrente.
  • Dificultad para administrar la seguridad: cada aplicación se crea independientemente; siendo muy difícil establecer criterios de seguridad uniformes.

2 Sistemas de información orientados a los datos (bases de datos):

los datos se centralizan en una base de datos común a todas las aplicaciones.
