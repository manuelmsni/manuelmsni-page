Diagrama base de datos de una biblioteca ⬇️

Crea el diagrama de una base de datos para la gestión de una biblioteca, en ella se ha de poder:

  1. Dar de alta, modificar y dar de baja libros.
  2. Dar de alta prestamos (teniendo en cuenta que puede haber varios ejemplares de cada libro) diferenciando si el prestamo ha sido realizado por un alumno o un profesor. Así como las devoluciones.
  3. Dar de alta, modificar y dar de baja lectores.
